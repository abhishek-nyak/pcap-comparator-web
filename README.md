# PCAP Comparator Web App

A pure-Python web application built with Streamlit and PyShark to perform order-agnostic comparisons of HTTP streams between two PCAP files.

## Local Setup
1. Clone the repo: `git clone <your-repo-url>`
2. Create virtual environment: `python -m venv venv`
3. Activate environment and install dependencies: `pip install -r requirements.txt`
4. Ensure `tshark` is installed on your OS.
5. Run the app: `streamlit run app.py`
